# AI-Assisted Smart Meeting Scheduler

## Overview
AI-Assisted Smart Meeting Scheduler is a web-based application that helps users automatically find and book the best meeting times. The system shows available slots, ranks them, and sends email confirmations after booking.

---

## Problem Understanding

Scheduling meetings manually is difficult because different people have different availability times. This project solves this problem by automatically finding the best meeting slots and allowing users to book meetings easily.

Users can:

- Log in using email
- View available slots
- Select a meeting slot
- Book meetings
- Receive email confirmation

## System Flow

1. User enters email and logs in
2. Available meeting slots are displayed
3. Slots are ranked automatically
4. User selects a slot
5. Meeting is booked
6. Confirmation email is sent

